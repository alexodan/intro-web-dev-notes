## HTML

<b>HyperText Markup Language</b>, como dice el nombre (y en informatica practicamente todo esta en ingles asi que a acostumbrarse), es un lenguaje de marcado con el que le decimos a los navegadores web (Chrome, Firefox, Safari, Edge...) como estructurar las paginas web, es decir el esqueleto del sitio, titulos, encabezado, pie de pagina, parrafos, formularios, etcetera.

Esto es codigo HTML:
```html
<h1>Esto es un titulo</h1>
<p>Esto es un parrafo</p>
<ul>
  <li>Primer item</li>
  <li>Segundo item</li>
  <li>Tercer item</li>
</ul>

```

Al ser interpretado por un navegador se ve mas o menos asi:

<h1>Esto es un titulo</h1>

<p>Esto es un parrafo</p>

<ul>
  <li>Primer item</li>
  <li>Segundo item</li>
  <li>Tercer item</li>
</ul>
