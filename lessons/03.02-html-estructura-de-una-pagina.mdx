Como dijimos anteriormente, el contenido visible de nuestro sitio web se va a
ver dentro de la etiqueta `<body>`.

Existen etiquetas que son "estructurales", quiere decir que el proposito que
sirven es de separar y organizar el contenido de la pagina web para que se le
haga mas facil a cualquier persona que lea el codigo HTML y pueda ubicarse. Es
decir que el usuario final que ve el resultado final de la pagina web en su
navegador, estas etiquetas no le cambian en nada su experiencia o como ven la
pagina.

Estas etiquetas que llamo estructurales son: `<div>`, `<main>`, `<footer>`,
`<header>`, `<aside>` y creo que son todas.

Veamos <a href="">este ejemplo</a> en codesandbox, ahi estoy usando header, main
y footer para ordenar el contenido del sitio, sin embargo de nuevo, aunque esas
etiquetas no estuviesen ahi el resultado final seria el mismo, pero es
importante usarlas para separar y organizar mejor el contenido de nuestro sitio,
y va a adquirir incluso mayor importancia cuando empecemos a agregarle estilo al
sitio web (espaciado, colores, animacion, etc.).

#### Ejercicio 1

Hace la prueba y borra las etiquetas header, main y footer del sandbox de
ejemplo, y fijate como queda la pagina web, notas alguna diferencia? Una vez
borradas esas etiquetas deberia quedar <a href="">algo asi</a>.

#### Ejercicio 2

Como ejercicio create un nuevo sandbox (con el boton "create" que vimos antes),
y hace un sitio web que tenga un titulo, una lista desordenada con las tareas
que tengas para hacer y otra lista pero esta vez ordenada (`<ol>`) con tu top 5
de peliculas o series favoritas o lo que se te antoje.
