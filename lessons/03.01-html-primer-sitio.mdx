Para que podamos empezar a desarrollar sitios web, debemos tener un editor de texto. Si bien uno tan basico como notepad (Windows) o gedit (Linux) nos puede servir, hay editores especializados para desarrolladores que nos dan muchisimas mas herramientas.

Antes de pasar por el proceso de instalar uno, vamos primero a utilizar uno online, llamado <a href="https://codesandbox.io/">codesandbox</a>.
