Ya que vamos a aprender sobre desarrollo web,
probablemente sea una buena idea tener un poco mas claro de que se trata este concepto de <b>internet</b> y como funciona, este videito de youtube esta bueno para ver un poco de eso: https://www.youtube.com/watch?v=u1xxZ8r2rRc

Tambien <a href="https://developer.mozilla.org/es/docs/Learn/Common_questions/How_does_the_Internet_work#encontrando_ordenadores">este articulo</a> sobre como funciona internet de <b>MDN (mozilla developer network)</b> no esta de mas pegarle una leida, principalmente el parrafo de <em>"Encontrando ordenadores"</em> e <em>"Internet y la web"</em>.

Y por ultimo <a href="https://developer.mozilla.org/es/docs/Learn/Getting_started_with_the_web/How_the_Web_works">este articulo</a> (completo) de como funciona la web, tambien de MDN (este sitio lo vamos a usar mucho ya que tiene excelente info).

Ese ultimo articulo es un toque intimidante pero tranka, vamos a ver con mas detalle la comunicacion cliente/servidor con el paso del tiempo, lo importante es tener una vision general e ir incorporando conceptos de a poco.
